# 😴 Should I Take a Nap?

> A highly scientific, peer-unreviewed nap necessity assessment tool.

A short quiz that asks you 6 deeply important questions about your current level of consciousness and delivers a dramatic verdict on whether you should take a nap.

## Live Demo

Just open `index.html` in any browser — on your phone, laptop, or the screen you're currently struggling to focus on.

## Features

- 6 questions of rigorous sleep-science research (vibes-based)
- 4 possible verdicts ranging from "You're fine" to "You are already asleep"
- Nap necessity score bar for data-driven decision making
- Fully responsive — works on mobile
- Zero dependencies, zero npm install, zero effort

## How to Run

```bash
git clone https://github.com/cfcd3/should-i-nap.git
cd should-i-nap
open index.html   # or just double-click it
```

That's it. It's one HTML file.

## Verdicts

| Score | Verdict |
|-------|---------|
| 0–4   | No Nap Needed |
| 5–9   | Nap Curious |
| 10–14 | Nap Immediately |
| 15–18 | You Are Already Asleep |

## Why

Because sometimes you need a quiz to give you permission to do what you were going to do anyway.
